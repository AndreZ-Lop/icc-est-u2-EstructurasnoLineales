## 1. Descripcion
Durante esta practica se realizaron distintos ejercicios que tenian una fuerte relacion a los recorrido de los arboles


## Ejercicio1
## Descripción:
El ejercicio 1 fue crear un arreglo de enteros para envase a este arreglo generar un arbol donde la root empieza en 5, y se va bifurcando con condiciones generando a los hijos con respetco a si el numero ingresado  es menor o mayor al root. El metodo para imprimir utilizado se base en el recorrido InOrder para imprimir el arbol en forma de piramide.

## Ejercicio2
## Descripción:
El ejercicio 2 se solicita invertir un arbol tal como en el anterior ejercicio se utilizo una lista de enteros para generar el arbol posterior a ello se utilizo el recorrido PreOrder para cambiar las referencias del arbol aplicando las estregia de guardar uno de los nodos en una variable auxiliar y para imprimirlo se utilizo el mismo metodo el ejercicio 1.

## Ejercicio3
## Descripción:
El ejercicio 3 se pide una lista de listas que almacenen en su interior nodos con cada nivel representando una lista donde se utilizo colas y list para su solucion ademas de implementar ciclos while y for para ir iterando la cola generada e ir agregando los elementos y eliminando elementos conforme los hijos de algun nodo se agreguen a la misma proceso que no finalizara hasta que toda la cola creada para este proceso se vaciara por completo significando asi que todos los elementos del arbol fueron procesados.

## Ejercicio4
## Descripción:
El ejercicio 4 se aplico recursividad para su solucion donde se creo un metodo que recibe la raiz del arbol e iba calculando la profundidad maxima del mismo siendo por eso que se creo un baso base que retorne 0 marcando asi el final de este arbo, y llegados a este punto el caso recursivo ira retrocediendo hasta llegar a la raiz haciendo comparaciones para quedarnos unicamente con el mayor valor obtenido en el calculo de la altura.
